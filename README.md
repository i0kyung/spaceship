# 꿈돌이의 첫 비행

1993 대전엑스포 마스코트 꿈돌이의 세계관을 바탕으로, 사용자가 우주선을 타고 지구 대전으로 떠나는 여정을 스크롤로 경험하는 인터랙티브 스토리 웹 프로토타입입니다.

공개 페이지: https://i0kyung.github.io/spaceship/

## 주요 경험

- 닉네임 입력 + 손 인식(MediaPipe Hands) 등록으로 시작하는 탑승 준비 (카메라 실패 시 마우스 모드로 자동 전환)
- 마우스 위치에 따라 꿈돌이의 시선이 움직이는 첫 만남
- 손 또는 마우스로 뿌연 우주창을 닦는 Canvas 인터랙션 (진행률 게이지 포함)
- 짧은 로딩 연출을 거쳐 이어지는 목적지 뽑기 — 항상 지구/대전으로 수렴하는 캡슐 뽑기
- 워프 터널 끝에서 지구 도착을 보여주는 마무리 장면

## 빠른 시작

```bash
npm install
npm run dev
```

개발 서버가 실행되면 http://localhost:5173/ 에서 확인할 수 있습니다.

빌드 확인:

```bash
npm run build
```

구현 구조와 에셋 설명은 [프로젝트 설명](./HANDOFF.md)에 정리되어 있습니다.
